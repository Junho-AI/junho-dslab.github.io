## BERT?

![BERT](/assets/img/bert.png)
BERT 코드 리뷰


```python
import sys
try:
    total = sum(int(arg) for arg in sys.argv[1:])
    print 'sum =', total
except ValueError:
    print 'Please supply integer arguments'
```

2장에서 논의할 분포의 역할들 중 하나는 한정된 수의 관찰 집합 $x_{1},...,x_{N}$이 주어졌을 때 확률 변수 $x$의 확률 분포 $p(x)$를 모델링하는 것이다.
